id tags
This site contains information about some of the basic HTML tags.
Head Element

The head element goes at the top of the html file, before the body.

The head element contains other html elements that do not show up in the body of the webpage. Elements included in the head tag are listed below.

  • title
  • style
  • links
P tags

p tags contain descriptive information.

This element usually contains most of what you read on a website.

The id tag is used to assign a specific id to an element in your code.

The id is represented in CSS by a # followed by the name of the id. An example is given below.

  • id= blue-dot;
  • #blue-dot
Section tags

Sections of a document are defined by the section tag.

This tag can be used to define headers, footers, chapters, or any other section of the document.

Class tags

class tags operate similar to id tags, but they are less specific.

Class tags can be used to represent multiple elements, and when selected for in CSS are represented by a . followed by the class name.
